This is the list of valid urls to control the lights

Path Description 
{host}/ledon
 Turns on the LED light strip 
{host}/ledoff
 Turns off the LED light strip 
{host}/ledset?rgb=RRRGGGBBBWWW
 Set the LED color to the specified RGB value (values 0-255) with specified brightness (value 0-255) 
{host}/ledrainbow
 Set the LED color to fade though the colors of the rainbow
